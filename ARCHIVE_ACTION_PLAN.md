# Sima Berman Surname-Bridge Archive Action Plan

Date: 2026-04-15
Timezone: Asia/Jerusalem

## Purpose

This plan is designed to resolve the surname contradiction in the case of `Sima Berman`.

This is a **surname-bridge case**:
- official surname: `Krakowski`
- possible earlier surname: `Buxenbaum / Buxenboim / Nussbaum`

The goal is to determine whether these surname lines belong to the same woman and the same birth family.

## Case Summary

### Official-record anchor

- `Sima Krakowski`
- father: `Yitzhak-Meir Krakowski`
- mother: `Malka`
- marriage certificate: `23.12.1937`, `Afula`

### Family-memory line

- possible earlier surname:
  - `Buxenbaum`
  - `Buxenboim`
  - `Nussbaum`
  - `Nussenbaum`

### Additional anchors

- birth date: `24.12.1912`
- possible early-life location: `Kovel/Kowel`
- Sima immigrated to Palestine in `1935`
- she was connected to `Hashomer Hatzair`
- mother died young
- father remarried
- at least one brother existed

## Priority Order

1. `Polish Aliyah Passports / immigration records around 1935`
2. `Dukla Jewish Community Registration Records`
3. `Kovel archival/community records`
4. `Arolsen Archives` only if needed
