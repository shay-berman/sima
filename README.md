# Sima Berman Research Repo

This repository is the main tracking hub for the genealogy research into the birth family of `Sima Berman`.

## Main Goal

Identify the birth family of:

- `Sima Berman`
- later official surname: `Krakowski`
- possible earlier surname:
  - `Buxenbaum`
  - `Buxenboim`
  - `Nussbaum`

Current research focus:
- reconstruct the original family from `Kovel / Kowel / Volhynia`
- resolve the surname contradiction between `Krakowski` and `Buxenbaum/Nussbaum`
- identify siblings and surviving branches
- locate official pre-immigration records from `1935`

## Current Status

The strongest current findings are:

- official marriage record in Palestine identifies:
  - `Sima Krakowski`
  - father: `Yitzhak-Meir Krakowski`
  - mother: `Malka`
- family evidence strongly supports:
  - `Sima Berman (née Buxenbaum)`
  - born `24 Dec 1912`
  - born in `Kovel`
  - father: `Yerucham Meir Buxenbaum`
  - siblings:
    - `Frida`
    - `Sonia`
    - `Sara`
    - `Chaim`
- `JHI` has already replied once:
  - no passport found in the first pass
  - a second check is now in progress using `Kovel / Kowel` as the place-of-birth anchor
- internal Yad Vashem searching has identified a real `Boksenbojm` family branch in `Kowel`, but it is not yet proven to be Sima's immediate household

## Key Working Problems

### 1. Surname contradiction

Official records support:
- `Krakowski`

Family-memory and other evidence support:
- `Buxenbaum`
- `Buxenboim`
- `Nussbaum`

The central unresolved question is:

Did the same woman and family move between these surname lines before immigration?

### 2. Missing immediate family records

Still not proven directly in records:
- `Frida`
- `Sonia`
- `Sara`
- `Chaim`
- direct household of `Yerucham Meir Buxenbaum`

### 3. Kovel Boksenbojm cluster

The internal Yad Vashem interface surfaced a confirmed `Boksenbojm` branch in `Kowel`, including:
- `Pinchas Boksenbojm`
- `Rachel Boksenbojm`
- `Alter Boksenbojm` (1881)
- older `Alter Boksenbojm` (c.1858)

This is important because:
- it proves a real surname branch in `Kowel`
- it may represent the wider family network
- but it is not yet proven as Sima's immediate household

## GitHub Pages

This repository now includes a lightweight GitHub Pages site based on the repository documentation.

### Expected publish URL

- `https://shay-berman.github.io/sima/`

### Files used by the site

- `index.html`
- `docs.html`
- `style.css`
- `script.js`
- `.github/workflows/pages.yml`

The document pages load the Markdown files from this repository directly, so updates to the `.md` files will appear on the site after a new deployment.

### How to enable GitHub Pages in GitHub

1. Open the repository on GitHub.
2. Go to `Settings`.
3. Open `Pages`.
4. Under `Build and deployment`, choose `GitHub Actions` as the source.
5. Save if GitHub asks you to confirm the source.

After that, every push to `main` should trigger the Pages workflow and publish the site.

### How to update the site later

1. Edit the Markdown files or the site files in the repository.
2. Commit and push to `main`.
3. GitHub Actions will redeploy the Pages site automatically.

If you want to change the homepage content, edit:

- `README.md`
- `index.html`

If you want to change the styling, edit:

- `style.css`
